## Installation

```console
pip install djangocms-fomantic-ui
```

If you want to use the example template of this project, add it to `CMS_TEMPLATES`:

```python

    ("djangocms_fomantic_ui/fomantic-ui.html", "Fomantic UI"),
]

```
