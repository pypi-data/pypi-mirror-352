關於此服務

這是一個 Model Context Protocol (MCP) 服務器，用於在 AI 輔助開發工具中提供人在回路的互動回饋功能。

當 AI 助手需要用戶回饋時，會自動在瀏覽器中開啟互動頁面。

功能特色

  • 🌐 Web UI 支援 SSH remote 開發
  • 💻 即時命令執行和輸出顯示
  • 💬 結構化回饋收集
  • ⚙️ 專案特定的設定管理
  • 🔄 WebSocket 即時通訊
